# Portfolio Project Images

This directory contains images for each portfolio project.

## Required Images Per Project

For each project folder, you need to provide:

1. **thumbnail.jpg** - Portfolio grid card image (recommended 800x600px)
2. **hero.jpg** - Project detail page hero image (recommended 1920x1080px)
3. **gallery-1.jpg, gallery-2.jpg, gallery-3.jpg** - Case study screenshots

## Project Folders

- `slovak-luxury-living/` - Real estate website project
- `billik/` - E-commerce welding supplier project
- `slohgpt/` - AI essay grading SaaS
- `realfoto/` - Real estate photo enhancement SaaS
- `notewall/` - iOS lock screen productivity app
- `faithwall/` - iOS Bible verse app

## Image Guidelines

- Use high-quality JPG or PNG format
- Optimize for web (compress to reasonable file sizes)
- Maintain aspect ratios appropriate for project showcase
- Include app store screenshots, website screenshots, or app mockups as relevant
